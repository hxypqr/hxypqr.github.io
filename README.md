## üöÄ Running Locally

To run this website locally, you'll need to have Ruby and the Jekyll gem installed.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/hxypqr/hxypqr.github.io.git
    cd hxypqr.github.io
    ```

2.  **Install dependencies:**
    It's recommended to use Bundler to manage dependencies.
    ```bash
    bundle install
    ```

3.  **Serve the site:**
    ```bash
    bundle exec jekyll serve
    ```
    The site will be available at `http://localhost:4000`.
